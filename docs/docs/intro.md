# Welcome to SoundCore-Desktop Documentation

SoundCore-Desktop is a lightweight cross-platform tray app that restores your Soundcore device settings automatically when your device connects.

This docs website is built with Docusaurus and provides installation, usage, and deployment guidance for the project.

## What this project includes

- Automatic Soundcore profile restore for gaming mode, ANC, and EQ
- Native desktop tray controls on Windows and Linux
- OpenSCQ30-based Bluetooth device handling
- A Tauri + React desktop app architecture

## What you will find here

- **Installation**: prerequisites, dependency setup, and running the app.
- **Features**: what SoundCore-Desktop supports and how it improves your workflow.
- **Usage**: how to launch the tray app, control profiles, and restore settings.
- **Architecture**: project internals, modules, and folder structure.
- **Troubleshooting**: common problems and recovery steps.
- **Deployment**: docs and static site publishing information.
